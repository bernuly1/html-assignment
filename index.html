1.The differences between the web versions

  • web 1.0 It is the “readable” phrase of the World Wide Web with flat data. In Web 1.0, there is only limited interaction between sites and web users.
    • Has static pages and web sites. The search is not very accurate and update of the data is not so frequant.
    • These webs were "close source" and had a basic design.
    • Frames and Tables used to position and align the elements on a page
  • web 2.0 It is the “writable” phrase of the World Wide Web with interactive data. Unlike Web 1.0, Web 2.0 facilitates interaction between web users and sites, so it allows users to interact more freely with each other.
    • Supports social sites such as MySpace and facebook in addition to blogs and micro blogs for example Twitter.
    • Dynamic content that is responsive to user input (wikis) and share content like youtube.
    • Web access leads to concern different, from the traditional Internet user base to a wider variety of users.
  • web 3.0 It is the “executable” phrase of Word Wide Web with dynamic applications, interactive services, and “machine-to-machine” interaction.
    • Uses AI algorithms to discover new trends on web.
    • Encludes Cognitive computaion, machne learning and NLP.
    Use case


    A user wants to search a guitar shop.

    Results:
  • Web 1.0 The search results would probably be sites that have guitars (not necessarily selling them) and maybe shops that don’t have guitars. It will also show some links to guitar shops, not necessarily next to the user.
  • Web 2.0 The search results would probably be sites about guitar shops and links to sites that connected to the guitar shops.
  • Web 3.0 The search results will have elements that react to information about the user. Guitar shops that are located next to the user will appear first as well as pictures and social pages of the shops.
